# clash

## Fast Setup

You need [npm] and [Node.js] installed on your machine

### Game Server
Installation
```sh
$ npm install
```
run server
```sh
$ node app.js
```

### Client Server
Installation
```sh
$ npm install
```
run server
```sh
$ node app.js
```

   [Node.js]: <https://nodejs.org/en/>
   [npm]: <https://www.npmjs.com/>
